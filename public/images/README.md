# Image Assets

Place the following assets in these folders:

## /hero/
- Hero product images from Figma (Desktop - 18 hero image)
- Recommended: Export as PNG or WebP at 2x scale
- Suggested filename: `hero-main.png`

## /catalog/
- Catalog page images (exported from your PDF)
- Name them sequentially: `page-01.png`, `page-02.png`, etc.
- Or organize by category if preferred

## Root folder (here)
- `logo.svg` - Export the Logo layer from Figma as SVG
